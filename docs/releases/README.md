## 🚀 升级指南

### 从 v1.6.x 升级到 v1.7.x

v1.7.x 版本保持了 100% 的向后兼容性，可以直接升级。

**升级步骤**：

1. **备份数据**（可选）

   ```bash
   # 备份配置和数据库
   cp -r ~/.plooking ~/.plooking.backup
   ```

1. **下载新版本**

   - 从 [Releases](https://github.com/onlyhooops/plookingII/releases) 下载最新版本
   - 或使用包管理器更新

1. **安装**

   - 替换旧版本应用
   - 首次启动会自动迁移数据

1. **验证**

   - 检查图片浏览功能是否正常
   - 确认历史记录和精选已迁移
   - 验证快捷键和设置

**注意事项**：

- ✅ 所有 API 保持兼容
- ✅ 配置文件格式不变
- ✅ 数据库自动迁移
- ⚠️ 建议先备份重要数据

## 📝 版本命名规范

PlookingII 遵循 [语义化版本](https://semver.org/) 规范：

```
主版本号.次版本号.修订号 (MAJOR.MINOR.PATCH)

例如: 1.7.1
- 1: 主版本号（重大不兼容更改）
- 7: 次版本号（新增功能，向后兼容）
- 1: 修订号（Bug 修复，向后兼容）
```

### 版本规则

- **MAJOR** (主版本号): 不兼容的 API 更改
- **MINOR** (次版本号): 向后兼容的新功能
- **PATCH** (修订号): 向后兼容的 Bug 修复
