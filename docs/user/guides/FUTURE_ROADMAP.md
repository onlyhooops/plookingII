# Plooking 开发规划

## v1.4.0
- 新增功能
1. 支持通过SMB挂载的远程盘符并对性能进行专项优化
    - 网络与磁盘I/O优化
    - 缓存机制优化
    - 改善浏览性能
2. 支持视频资源的浏览
    - 支持常规格式的视频资源浏览/预览，目标视频分辨率至高为4K
        - MP4
        - MOV
    - 进行针对性的性能优化
    - 内建播放器（可集成系统级预览组件），有视频进度控制条并能拖动条件进度
- 性能调优
1. 持续优化超高分辨率竖向图片的加载/渲染性能，提升浏览/切换流畅性
2. 持续优化连续横向/竖向图片后加载竖向/横向图片时的性能（两种方向图片切换时的渲染性能）
3. 优化超高分辨率照片加载/切换性能
4. 增加预加载队列，从前后3张增加到前后5张
